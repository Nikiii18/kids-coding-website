Coding Platforms

Code.org

Code.org is a nonprofit with a catalog of courses they've designed, and some from 3rd party partners as well, to help guide students in grades K-12 on their coding path.

Scratch

Scratch is a platform designed by MIT that is a great way for children to learn coding. It's a free programming language and online community where you can create your own interactive stories, games, and animations.

W3Schools

W3Schools is a freemium educational website for learning coding online. For students in middle school and high school, W3Schools is a great resource for coding tutorials in multiple languages..

Codingal

Codingal is a community-driven mobile and web-based platform for kids to learn to code together..

CodeAvengers

Code Avengers is one of the best online coding platforms that involve mathematics, logical operations and intellectual ways to create new programs..

Blockly

Blockly teaches programming through puzzles and block games. Kids can create their own stories by clicking and dragging different blocks and solve problems..
